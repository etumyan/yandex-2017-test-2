# Yandex Task 2

## Установка

`git clone git@github.com:etumyan/yandex-2017-test-2.git`

`cd yandex-2017-test-2`

`npm install`

## Структура данных

```
Lectures
|-- name: string
|-- reader: string
|-- startDate: date
|-- endDate: date
|-- Room: reference

Schools
|-- name: string
|-- studentNumber: integer

Rooms
|-- name: string
|-- capacity: integer
|-- description: string
```
